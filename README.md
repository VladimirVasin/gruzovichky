# Gruzovichky

`Gruzovichky` is a simple Unity 3D transport prototype built around a small grid-based logistics loop.

The player places roads on a visible 2.5D grid and connects key locations: `Parking`, `Forest`, `Warehouse`, and `Town`. A single truck lives at the parking area, drives only along connected road cells, picks up wood in the forest, delivers it to the warehouse, and then carries it onward to the town.

The project is intentionally lightweight and prototype-focused:

- low-poly placeholder visuals
- one truck
- one resource (`wood`)
- simple automatic transport logic
- basic camera controls and minimal UI

The current goal of the project is to validate the core gameplay loop before adding more systems, vehicles, or economic complexity.
